海角晓彤表妹移花宫

这个人，他真的是明星吗？？
　　“大家帮我做个证啊，我是反扒，不是扒手。”
　　“等待会儿工作结束，我会把手机全都上交的，保证全程直播！”
　　林夏这会儿的求生欲可谓是拉满了。
　　疯狂跟直播间解释着。
　　而直播间的观众们看到这一幕。
　　一个个笑的跟吃屁一样，肚子都抽抽了。
　　“噗哈哈！神特么顺手。”
　　“反扒？你确定你这是反扒？不是顺手捞点外快嘛！”
　　“细思极恐，如果主播刚才不告诉我们，谁能知道他已经顺了两个手机？”
　　“嘶？这踏马？简直让人头皮发麻啊！”
　　“所以，这个逼真的是明星吗？他为什么要当明星？”
　　“按照他的说法，他主业确实是明星艺人，这每天的兼职只不过是他的副业罢了。”
　　“副业超过主业系列是吧？秀儿啊！”
　　“……”
　　就在直播间观众们议论的时候。
　　林夏好像看到了一个熟悉的弹幕。
　　不过水友们没注意，林夏倒是也没多说。
　　只是多强调了一遍，等结束他会上交的。
　　接着，他调转目光，开始寻找下一个扒手。
　　……
　　而与此同时，
　　另一边。
